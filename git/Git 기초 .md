### Git 기초 

1. git 설정 확인<br>
	```
	git config --list 
	```

2. git 저장소 만들기
	* 다른 저장소를 clone
	* 버전 관리를 하지 않는 디렉토리를 git 저장소로 만든는 방법<br>
	``` 
	git init 
	```
	
3. git 상태 확인<br>
	```
	git status
	```
